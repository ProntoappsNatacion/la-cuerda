Decisión arquitectónica clave

El parseo de xlsx se hace 100% en el navegador con SheetJS — no se sube ningún archivo a ningún servidor. El sistema extrae los datos y los guarda en Supabase. Rápido, seguro, sin costos de storage.

03 · Fuentes de datos externas (archivos xlsx)

📄 OlaClick — Hoja: Productos

Subida diaria o periódica. La hoja PRODUCTOS es la clave: tiene el acumulado de unidades y dinero por producto.

  • Nombre de la variante de producto→ mapear a productos RestoApp
  • Cantidad vendida→ sumar a estadísticas
  • Total vendido→ ingresos del período
  • Ganancia por unidad→ margen por producto
  • Categoría→ filtros estadísticos
  • Hoja Pedidos → Fecha→ ventas por día
  • Hoja Pagos → Método pago→ cuadre de caja
📄 Rappi — Hoja: Detalle

Subida semanal (pago semana vencida). Uso: SOLO estadístico. Los descuentos → gastos automáticos.

  • Venta Bruta→ ingreso bruto Rappi
  • Descuento asumido por aliado (20%)→ gasto automático
  • Uso y alquiler plataforma Rappi→ gasto automático
  • IVA + Retefuente Rappi→ gasto fiscal
  • Fecha de creación orden→ día más vendido
  • Estado de la orden→ filtrar canceladas
  • Valor Neto final→ lo que realmente paga Rappi

→

→

→

06 · Flujos críticos — Lógica de negocio

🛒 Al vender en POS

// 1. Crear venta INSERT INTO sales (source='pos', total, payment_method_id, sale_date, is_credit=false) // 2. Crear items INSERT INTO sale_items (product_id, quantity, unit_price, unit_cost) // 3. Descontar inventario (si la receta está configurada) FOR EACH recipe_ingredient WHERE product_id = vendido: UPDATE ingredients SET stock_current -= (quantity * items_sold) // 4. Verificar alertas de stock bajo IF stock_current <= stock_minimum → NOTIFY alerta_bajo_stock

🧾 Al registrar factura de compra

// 1. Crear compra con items INSERT INTO purchases + purchase_items // 2. Para cada ingrediente comprado: prev_cost = ingredients.cost_per_unit new_cost = purchase_item.total_cost / purchase_item.quantity UPDATE ingredients SET stock_current += quantity, cost_previous = prev_cost, cost_per_unit = new_cost // 3. Alerta si el costo cambió IF ABS(new_cost - prev_cost) / prev_cost > 0.05: // más del 5% NOTIFY "El costo de [ingrediente] subió/bajó X%" // 4. Recalcular costo de todas las recetas que usan ese ingrediente FOR EACH recipe_ingredient WHERE ingredient_id = comprado: recalculate recipe.total_cost update product.cost // 5. Crear gasto en Finanzas (tipo: purchase) INSERT INTO expenses (type='purchase', amount=total_factura, purchase_id)

🏍️ Al importar Rappi xlsx

// 1. SheetJS lee hoja "Detalle" en el browser // 2. Calcula totales venta_bruta = SUM(rows["Venta Bruta"]) comision = SUM(rows["Descuento de Producto asumido por el aliado"]) tarifa = SUM(rows["Uso y alquiler de plataforma Rappi"]) valor_neto = venta_bruta + comision + tarifa // comision es negativo best_day = rows.groupBy(dayOfWeek).max() // 3. Guardar importación INSERT INTO rappi_imports (venta_bruta, comision_20pct, tarifa, valor_neto, best_day) // 4. Crear venta a CRÉDITO (productos cuentan, dinero no hasta pago) INSERT INTO sales (source='rappi', total=venta_bruta, is_credit=true, credit_paid_at=null) // 5. Crear gastos automáticos (comisiones y tarifas) INSERT INTO expenses (type='rappi_fee', description='Comisión Rappi 20%', amount=comision) INSERT INTO expenses (type='rappi_fee', description='Tarifa plataforma Rappi', amount=tarifa) // 6. Cuando llega el pago (usuario lo marca): UPDATE sales SET credit_paid_at = NOW() // recién ahí entra a Finanzas UPDATE rappi_imports SET paid_at = NOW()

📄 Al importar OlaClick xlsx

// 1. SheetJS lee hoja "Productos" (la más importante) products_sheet = workbook["Productos"] // 2. Para cada producto en el xlsx FOR EACH row in products_sheet: // Buscar mapeo: olaclick_name → product local local_product = products.find(olaclick_name = row["Nombre de la variante de producto"]) IF local_product found: // Crear sale_items para estadísticas INSERT INTO sale_items (product_id, quantity=row["Cantidad vendida"], unit_price=row["Ganancia por unidad"]) ELSE: // Mostrar UI: "Mapear producto: 'Tres Leches -' → ¿cuál producto tuyo?" flag_for_mapping(row["Nombre"]) // 3. Leer hoja "Pagos" para cuadre de caja FOR EACH row in pagos_sheet: totals_by_method[row["Método de pago"]] += row["Monto total recibido"] // 4. Crear venta consolidada (source='olaclick') INSERT INTO sales (source='olaclick', total=SUM, is_credit=false)
